Une application multiplatforme pour aider à gérer ses dépenses.


Pense (dé**pense**s) est une application de prise de note pour vos dépenses mensuelles.

L'application à été crée car j'avais le besoin d'une application qui permettrai de m'aider a traquer mes dépenses et que j'avais la motivation de la faire moi-même.  

Pense possède les fonctionnalités suivantes : 

- Stocke vos dépenses et revenus pour chaque mois.

- Différents graphiques pour aider à visualiser les données

- Statistiques variées pour les dépenses et les revenus

- Organise les données en catégories

- Rétrospective de l'activité sur une période donnée

- Thème et couleurs customisable

Vous pouvez télécharger l'application pour votre système (Android , Windows ou Linux) si vous souhaitez l'essayer en cliquant [ici](https://github.com/sylanecpn/pense/releases/latest).

Vous pouvez également jetter un œil à la [page Github du projet](https://github.com/sylanecpn/pense).
