Conception d'une application permettant l'automatisation du lancement et du traitement des test des antennes de télécommunication via la plateforme Sirius.

Conception d'un script Python ayant pour but de pouvoir lancer et récuper les résultats des test de la plateforme de test Sirius depuis Jira.

Depuis une page Jira contenant l'ensemble des tests à lancer sur Sirius, l'utilisteur pouvait déclancher le programme à l'aide d'un boutton qui venait déclancher un pipeline Gitlab contenant le script. Le script se servait des informations de la requête pour extraire les tests depuis Jira et les identifier pour lancer leurs équivalents sur Sirius par appels d'API. 
Une fois les tests lancés, le programme attend la fin de l'exécution des tests pour récupérer et traiter les résultats avant de mettre à jour avec les informations obtenues,la page Jira initiale.
